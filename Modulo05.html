Módulo 05 - Criando Listas

Lista e umma coleção de itens relacionados. Você usa uma lista para organizar dados com uma sequência de passos ou para listar itens em um grupo.

O HTML possui dois tipos principais de lista:

  • Numeradas
  • Não numeradas

Para criar uma lista ordenada, utilizamos duas tags:

  1. <ol></ol>: que abre e fecha o tipo de lista
  2. <li>: abre cada item da lista

Para criar uma lista não ordenada, utilizamos duas tags:

  • <ul></ul>: que abre e fecha o tipo de lista
  • <li>: abre cada item da lista

Também é possivel criar combinaçoes de listas em uma página web. Essa combinaçõ é chamada de lista intercalada.

  1. Orange
    • Tela sensivel ao toque de 13
    • 3 entradas USB
  2. Lemon
    • Tela sensivel ao toque de 15
    • 2 entradas USB

Após a análise o grupo deverá escolher a melhor opção para o trabalho a ser realizado.

Personalizando listas

O atributo TYPE pode especificar o tipo de marcador em uma lista não ordenada, ou tipo de número em uma lista ordenada. Observe, abaixo:

Sintaxe <ol type="estilo do número">

  1. A: Para exibir letras maiúsculas.
  2. a: Para exibir letras minúsculas
  3. I: Para exibir números romanos grandes
  4. i: Para exibir números romano pequenos

Sintaxe <ul type="estilo do número">

  1. disc
  2. circle
  3. square
