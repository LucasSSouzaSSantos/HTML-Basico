Parâmetros dentro da tag form:

  • Name - Nome do Formulário
  • Metrod - Método de passagem dos dados
    1. GET - método não tão discreto, pois mostra os dados na barra de passagem
    2. POST - método mais discreto, pois não mostra os dados na barra de passagem
  • action - nessa configuração eu informo quem irá receber os dados.

tags usadas dentro da tag form

  • Label: Nome para indentificar
  • input:
    1. type: Informa o tipo de dado que vai ser passado
    2. name: O nome dessa tag
    3. size: O tamanho da caixa de texto
    4. maxlength: A quantidade máxima de caractere que vai poder ser digitado.
